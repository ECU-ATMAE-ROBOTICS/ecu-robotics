How to Do a Pull Request on GitHub

  1. Fork the Repository

    Start by navigating to the repository you want to contribute to on GitHub. Click the "Fork" button in the top right corner of the repository page. This will create a copy of the repository in your own GitHub account.

  2. Clone the Repository

    After forking, navigate to your GitHub account where you'll find the forked repository. Click the "Code" button and copy the repository's URL. Open your preferred Git client or terminal and use the following command to clone the repository to your local machine: 

    git clone <repository-url>

  3. Create a New Branch

    Change to the repository's directory on your local machine using the following command: 

    cd <repository-directory>

    Create a new branch for your changes with a descriptive name using the following command: 

    git checkout -b <branch-name>

  4. Make and Commit Your Changes

    Make the necessary changes to the files in your local repository using your preferred text editor or IDE. Once you're satisfied with the changes, save the files and stage them for commit: 

    git add .

    Commit the changes with a meaningful commit message:

    git commit -m "Your commit message here"

  5. Push Your Changes

    Push your changes to your forked repository on GitHub using the following command: 

    git push origin <branch-name>

  6. Create the Pull Request

    Visit the original repository on GitHub and switch to the branch you just pushed. Click the "Compare & pull request" button next to your branch name. Add a descriptive title and detailed description for your pull request, and then click the "Create pull request" button.
