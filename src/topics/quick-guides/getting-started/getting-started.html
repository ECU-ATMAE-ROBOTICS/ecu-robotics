Code Setup

  • Git will be our version control. We will use it to manage changes in our code, especially between muliple developers
  • Github will be the platform that will host our repositories
  • Python is the language we use on our Raspberry Pi's, and for any computer visions or scripting needs
  • Arduino IDE Is the IDE we use to develop and flash our code onto our Arduinos. Arduino runs off C/C++. The IDE will include a compiler, and the standard Arduino Library, but not the standard C/C++ library
  • Visual Studio Code is an advanced text editor, and will be used for pretty much anything. As a text editor, it will need other tools to complete anything that isn't basically just writing text. We use it mostly for Python.
