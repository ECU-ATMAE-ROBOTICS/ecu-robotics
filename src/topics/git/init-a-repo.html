Initializing a Repository

To start using Git for a new project, you need to initialize a Git repository in the project's root directory.

To initialize a repository:

  1. Navigate to the project's root directory using the command line.
  2. Run the command git init. This creates a new Git repository in the current directory.
  3. Git will create a hidden .git directory that stores the repository's metadata and version history.

After initializing a repository, you can start adding files, making commits, and managing the project's history.
