Basic Git Commands

Git provides a set of basic commands that you can use to manage your repository and collaborate with others. Here are some commonly used Git commands:

  • git add: Add files to the staging area.
  • git commit: Create a new commit with the changes in the staging area.
  • git push: Push local commits to a remote repository.
  • git pull: Fetch changes from a remote repository and merge them into the current branch.
  • git branch: List, create, or delete branches.
  • git merge: Merge changes from one branch into another.
  • git checkout: Switch between branches or restore files from a specific commit.
  • git clone: Clone a remote repository to create a local copy.
  • git status: Show the status of the current repository.
  • git log: Display the commit history.
